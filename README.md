# 🌿 Organik Ürün Tanıtım Web Sitesi

Bu proje, **Internet Based Programming** dersi kapsamında geliştirilmiştir. Web sitesi üzerinden organik ürünler tanıtılmakta ve kullanıcılarla etkileşim sağlanmaktadır. Sistem hem kullanıcı hem de admin girişi içermektedir.

## 🔑 Giriş Türleri

### 👤 Kullanıcı Girişi:
- Ürünleri inceleyebilir.
- Admin'e mesaj gönderebilir.

### 🛠️ Admin Girişi:
- Yeni ürün ekleyebilir.
- Gelen mesajları görüntüleyebilir.
- Kullanıcıları görüntüleyebilir veya silebilir.

## 💡 Özellikler

- Kullanıcı ve admin için ayrı giriş ekranları
- Admin paneli üzerinden ürün yönetimi
- Veritabanına kayıtlı kullanıcıların listelenmesi
- Kullanıcıların admin’e mesaj gönderebilmesi
- Duyarlı (responsive) arayüz tasarımı

## 🛠️ Kullanılan Teknolojiler

- HTML  
- CSS 
- JavaScript  
- PHP  
- MySQL
